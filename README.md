# portal-cns-lab

## 项目简介

`portal-cns-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放的页面不针对任何特定域名或网站，而是以实验室方式整理和展示一些静态网页内容。项目的主要目的是提供一个集中存放 HTML 页面的空间，便于测试、展示和分享。

## 目录说明

```
portal-cns-lab/
├── pages/          # 存放各个独立 HTML 页面
├── assets/         # 页面引用的资源文件（CSS、JS、图片等）
├── README.md       # 本文件
└── LICENSE         # 许可证文件（如有）
```

- **pages/**：每个子目录或文件代表一个独立的 HTML 页面，命名清晰以便识别。
- **assets/**：全局或页面共享的资源文件，按类型或用途组织。

## 页面归档说明

仓库中的 HTML 页面以独立、自包含为原则，每个页面通常包含完整的 HTML 结构、样式和脚本（或引用仓库内的资源）。页面内容可能涉及各类静态展示、功能演示或实验性项目。

- 页面之间没有依赖关系，可单独访问。
- 如需新增页面，请在 `pages/` 下创建新目录或文件，并确保引用资源路径正确。
- 建议在页面内添加简单的使用说明或版本注释。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来修正错误、改进页面或添加新内容。
- 请保持目录结构清晰，避免冗余文件。
- 对于不再需要的页面，建议先通过 Issue 讨论再进行删除。
- 本仓库不承诺定期更新，但会尽力维护内容的可用性。

## 其他

- 本仓库仅用于静态页面归档，不涉及动态服务或后端逻辑。
- 使用者可自由浏览和引用页面，但需遵守仓库许可证（如有）或页面内的声明。
